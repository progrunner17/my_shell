# prosh 

prosh literally means "extremely cure" and is a short form of PROgrunner's SHell.  
This shell is originally made for syspro, a experiment class for system programing in the university.

## feature
- support redirect
  - >
  - <
- pipe
  - |
- some built-in commands

## built in commands
- exit
- fg
- bg
- jobs
